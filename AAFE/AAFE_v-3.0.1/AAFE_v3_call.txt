call:
java -Xms1024m -Xmx1024m -jar jSteganalyzer.jar -<Bezeichner1 > <Wert1 > -<Bezeichner2 > <Wert2 > ...
-<BezeichnerN > <WertN >

Bsp.: java -Xms1024m -Xmx1024m -jar jSteganalyzer.jar -file $infile -class $class

Parameter: 

Name der zu untersuchenden Audiodatei
Bezeichner: -file
Typ: String
Default: -
Beschreibung: Legt die zu untersuchende Audiodatei fest. Befindet sich die Audiodatei
nicht im selben Ordner wie jSteganalyzer, kann auch der komplette Pfad
angegeben werden. Wird dieser Parameter nicht angegeben, so wird das
Programm beendet.

Fensterbreite (engl. Window Size)
Bezeichner: -windowsize
Typ: Ganze Zahl
Default: 1024
Beschreibung: Gibt die Breite eines Fensters in Samples an. Die Fensterbreite muss
einer Potenz von zwei entsprechen, da die Berechnung des Spektrums
durch die FFT dadurch verbessert wird.

Überlappung (engl. Overlapping)
Bezeichner: -overlapping
Typ: Ganze Zahl
Default: 0
Beschreibung: Gibt die Überlappung in Samples an. Die Uberlappung darf nicht größer
oder gleich der Fensterbreite sein.

Hammingfenster
Bezeichner: -usehamming
Typ: Boolescher Wert
Default: 0
Beschreibung: Entscheidet, ob das Hammingfenster als Fensterfunktion verwendet werden
soll oder nicht.

Klassifikation
Bezeichner: -class
Typ: String
Default: "no cover"
Beschreibung: Gibt die Klasse der zu untersuchenden Audiodatei an. Dieser Parameter
ist Freitext und kann dementsprechend vom Nutzer gewählt werden.
Wird dieser Parameter nicht verwendet, so wird "no cover" als Klassifikation jedem Merkmalsvektor hinzugefügt.

Startsample
Bezeichner: -from
Typ: Ganze Zahl
Default: "0"
Beschreibung: Gibt an, ab welchem Sample mit der Analyse begonnen werden soll.
Wird dieser Parameter nicht angegeben, wird mit dem ersten Sample
begonnen.

Endsample
Bezeichner: -to
Typ: Ganze Zahl
Default: "Maximum"
Beschreibung: Gibt an, bis zu welchem Sample analysiert werden soll. Wird dieser Parameter
nicht angegeben, so wird bis zur Maximallänge des jeweiligen
Kanals analysiert. Wird ein Sample angegeben, welches größer als die
Maximallänge des jeweiligen Kanals ist, so wird nur bis zum Maximum
analysiert.
